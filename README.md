# daralabs.com

The Daralabs AB website — a single self-contained static page. No build step,
no dependencies, no JS: `index.html` is the entire site.

Served via GitHub Pages. Deploy = push to `main`.

Content decisions, design spec and the change log live in the (private)
business repo under `daralabs/website-remake/`. Every claim on the page is
fact-verified there before it ships.
